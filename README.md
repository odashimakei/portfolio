# 自分用備忘録
ローカルでブランチを作成
$ git branch <ブランチ名>
ローカルでブランチを切り替え
$ git checkout <ブランチ名>
ブランチ作成 & 切り替え
$ git branch -b <ブランチ名>
ローカルの変更を確認する
$ git status
commitの変更履歴をみる
$ git log
リモートにプッシュ
$ git push origin <ブランチ名>
